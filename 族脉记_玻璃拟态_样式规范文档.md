# 族脉记 - 玻璃拟态UI风格规范文档

## 一、整体UI风格概述

### 1.1 设计风格定位
**玻璃拟态 (Glassmorphism)** - 一种现代、轻盈、通透的界面设计风格

### 1.2 设计理念
- **通透感**：通过半透明背景和模糊效果营造玻璃质感
- **轻盈感**：使用柔和的色彩和渐变，避免沉重感
- **温暖感**：采用青色系主色调，传递家族温情
- **现代感**：圆角设计、阴影层次、流畅动画

### 1.3 核心特征
| 特征 | 描述 |
|------|------|
| 透明度 | 背景使用 rgba(255,255,255,0.6-0.85) 的半透明效果 |
| 模糊 | backdrop-filter: blur(10-20px) 营造毛玻璃效果 |
| 边框 | 1px solid rgba(255,255,255,0.4-0.6) 细腻边框 |
| 阴影 | 多层阴影营造深度感 |
| 渐变 | 青色系渐变作为主色调 |

## 八、设计原则总结

1. **一致性**：所有卡片使用统一的glass-card样式
2. **层次感**：通过阴影、透明度、模糊营造深度
3. **温暖感**：青色系主色调传递家族温情
4. **简洁性**：避免过多装饰，聚焦内容
5. **可交互性**：清晰的hover状态和点击反馈
6. **可读性**：合适的对比度，清晰的文字层级

# 族脉记 - 温暖家族风格UI规范文档

## 一、整体UI风格概述

### 1.1 设计风格定位
**温暖家族风格** - 以暖棕色为主色调，营造温馨、传统、亲切的家族氛围

### 1.2 设计理念
- **温暖感**：暖棕色/米色主色调，传递家的温暖
- **传统感**：简洁的卡片设计，不失现代感
- **亲切感**：圆润的边角，柔和的阴影
- **专业感**：清晰的层级，统一的规范

### 1.3 核心特征
| 特征 | 描述 |
|------|------|
| 主色调 | 暖棕色 #8B5E3C，象征家族的根基与传承 |
| 背景色 | 暖白色 #F8F6F3，营造温馨氛围 |
| 卡片 | 纯白背景 + 细边框 + 柔和阴影 |
| 圆角 | 统一的圆角设计，12-24px |
| 阴影 | 柔和的投影，营造层次感 |

```css

    transition: all 0.2s ease;

    transform: translateY(-1px);

    transform: translateY(0) scale(0.98);
```

### 5.1 选中动画 (数据移动上去之后)

当用户选中某个数据项时（如选择模板、选择标签、选择成员），触发以下动画：

```css

    }

        transform: scale(1.05);
        box-shadow: 0 4px 16px rgba(139, 94, 60, 0.3);
    }
    60% {
        transform: scale(1.02);
        box-shadow: 0 3px 12px rgba(139, 94, 60, 0.2);
    }
    100% {
        transform: scale(1.02);
        box-shadow: 0 3px 12px rgba(139, 94, 60, 0.2);
    }
}

.item-selected {
    animation: itemSelect 0.4s cubic-bezier(0.4, 0, 0.2, 1) forwards;
    background: #F5EDE3;
    border-color: #8B5E3C;
}

/* 取消选中动画 */
@keyframes itemDeselect {
    0% {
        transform: scale(1.02);
        box-shadow: 0 3px 12px rgba(139, 94, 60, 0.2);

    animation: itemDeselect 0.3s ease forwards;
}
```

## 八、设计原则总结

1. **一致性**：所有卡片使用统一的圆角、阴影、边框规范
2. **温暖感**：暖棕色主色调贯穿整个应用
3. **层次感**：通过阴影和间距营造清晰的视觉层级
4. **反馈感**：丰富的交互动画，让用户感知操作结果
5. **简洁性**：避免过多装饰，聚焦内容本身
6. **可用性**：清晰的选中状态，直观的操作反馈
